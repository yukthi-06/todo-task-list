### 3. File Operations

#### [MODIFY] [JsonManager.java](file:///x:/Projects_X/0_Active/8_Android_APK/02_Stable/TODO_Tasks_List_APK_GITHUB/app/src/main/java/com/vypeensoft/todo/JsonManager.java)
- Add a helper method `renameList(TodoList list, String oldName)` to rename a master list's JSON file on disk, avoiding duplicate files when a list is renamed:
  ```java
  public static void renameList(TodoList list, String oldName) {
      File dir = getDirectory();
      String oldFileName = oldName.replaceAll("[^a-zA-Z0-9.-]", "_") + ".json";
      File oldFile = new File(dir, oldFileName);
      
      saveList(list);
      
      String newFileName = list.name.replaceAll("[^a-zA-Z0-9.-]", "_") + ".json";
      if (!oldFileName.equals(newFileName) && oldFile.exists()) {
          oldFile.delete();
      }
  }
  ```
