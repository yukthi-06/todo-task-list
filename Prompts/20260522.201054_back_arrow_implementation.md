# User Prompt
```
Once a master list is opened, provide a left arrow on top to go back to master list
```

# Implementation Plan: Left Arrow to Go Back to Master List

We need to provide a back/left arrow when a master list (a todo list detail view) is opened, allowing the user to easily navigate back to the main master lists view.

To achieve this seamlessly, we will implement **two complementary paths**:

1. **Toolbar Back Navigation (Standard Android System Behavior)**:
   - Listen to backstack changes in `MainActivity`.
   - When a fragment is on the backstack (i.e. `ListDetailFragment` is opened), we hide the hamburger menu and display the standard Android back navigation arrow in the Toolbar.
   - Wire the back arrow click listener to pop the fragment backstack (`onBackPressed()`).
   - Lock the navigation drawer so that swiping is disabled while viewing details, conforming to standard Material guidelines.
   - When returning to the top level (no fragments on the backstack), restore the hamburger menu and unlock the drawer.

2. **Fragment Header Back Button (Layout-Level UI Navigation)**:
   - Update `fragment_list_detail.xml` to include a beautiful back arrow button (ImageButton) to the left of the list title `textTitle`.
   - Create a clean vector drawable `ic_back_arrow.xml` in `app/src/main/res/drawable/` containing the standard Material Design back arrow path.
   - In `ListDetailFragment.java`, wire this back button's click listener to trigger the Fragment Manager's `popBackStack()`.

This dual approach ensures:
- **Maximum UI visibility & ease of use**: Users can click either the top-left toolbar arrow or the list-level back arrow.
- **Flawless UI Integration**: Seamless transitions between navigation drawer state and detail view state.
- **Robust compatibility**: Works perfectly on all screen sizes and configurations.

## Steps

### Step 1: Create the Back Arrow Vector Asset
Create `app/src/main/res/drawable/ic_back_arrow.xml` with standard Material `arrow_back` path.

### Step 2: Modify `fragment_list_detail.xml`
Wrap `textTitle` in a horizontal `LinearLayout` and add the back `ImageButton` pointing to `ic_back_arrow`.

### Step 3: Implement Back Button Logic in `ListDetailFragment.java`
Bind the `ImageButton` and set an onClickListener that pops the back stack.

### Step 4: Add Back Stack Listener in `MainActivity.java`
Add a listener in `MainActivity` to update the Toolbar navigation icon and drawer lock state based on fragment back stack changes.
