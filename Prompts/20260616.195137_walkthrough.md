# Walkthrough - Compact Master List Row Height

> In the master list, make the row height lesser and more compact.  If the text wraps around, those rows can be taller..

## Changes Made

### Layout
- Modified [item_master_list.xml](file:///x:/Projects_X/0_Active/8_Android_APK/02_Stable/TODO_Tasks_List_APK_GITHUB/app/src/main/res/layout/item_master_list.xml):
  - Reduced Card View margin horizontal to `6dp` and vertical to `2dp` (previously `4dp` all-around).
  - Reduced Card View corner radius to `6dp` (previously `8dp`).
  - Reduced the `LinearLayout` container's vertical padding to `6dp` and horizontal padding to `12dp` (previously `16dp` all-around).

These changes reduce the vertical space occupied by each row item from ~80dp down to ~64dp, offering a much more compact listing. Because we kept `android:layout_height="wrap_content"`, any rows that have text that wraps to multiple lines will naturally expand.
