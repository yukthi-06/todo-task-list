# Compact Master List Row Height

> In the master list, make the row height lesser and more compact.  If the text wraps around, those rows can be taller..

## Proposed Changes

### [item_master_list.xml](file:///x:/Projects_X/0_Active/8_Android_APK/02_Stable/TODO_Tasks_List_APK_GITHUB/app/src/main/res/layout/item_master_list.xml)

Reduce the padding of the root container and card margins to make the list items much more compact. We will reduce the vertical padding of the item from `16dp` to `6dp`, and card margin from `4dp` to `2dp` (or use specific vertical margin).

#### [MODIFY] [item_master_list.xml](file:///x:/Projects_X/0_Active/8_Android_APK/02_Stable/TODO_Tasks_List_APK_GITHUB/app/src/main/res/layout/item_master_list.xml)

```xml
<<<<
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="horizontal"
        android:padding="16dp"
        android:gravity="center_vertical">
====
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="horizontal"
        android:paddingVertical="6dp"
        android:paddingHorizontal="12dp"
        android:gravity="center_vertical">
>>>>
```

And reduce the `MaterialCardView` margin:
```xml
<<<<

>>>>
```
